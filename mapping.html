3D Spatial Mapping System

3D mapping system implemented using a VL53L1X time-of-flight sensor, Python (using the Open3D library) , a stepper motor, and a MSP432E401Y microcontroller. The project also features the use of C, assembly language, the Cortex M4F, I2C, and UART.
