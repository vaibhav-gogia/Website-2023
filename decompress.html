3D Spatial Mapping System

• Hardware image decompression system created using System Verilog in the Quartus Prime environment.

• Includes design and implementation of 6-tap finite impulse response filter for horizontal up-sampling, colour space conversion operations from the YUV plane to the RGB plane, matrix multiplication operations, and use of embedded and external RAM units.

• Key design constraints included a finite number of hardware multipliers and external RAM units, a minimum operating clock frequency, and a minimum multiplier usage percentage.
